# Customer-Behavior-DataAnalyst-Portfolio-Project


## 📌 Project Overview

This project analyzes customer shopping behavior using Python, SQL, and Power BI. It demonstrates a complete data analysis workflow — from raw data processing to business insights and dashboard visualization.

The objective is to understand customer spending patterns, product performance, and the impact of discounts and subscriptions.

---

## 🧰 Tools & Technologies

* **Python (Pandas)** – Data cleaning and preprocessing
* **SQL (MySQL / PostgreSQL / SQL Server)** – Data analysis
* **Power BI** – Data visualization and dashboard creation

---

## 📂 Dataset

* 3900 customer records
* Includes demographics, purchase behavior, product details, and review ratings

---

# 🐍 Python Data Processing

### 🔹 Data Loading & Exploration

* Loaded dataset using Pandas
* Performed initial exploration using summary statistics and data types
* Identified missing values and data distribution

---

### 🔹 Handling Missing Values

* Found missing values in **Review Rating** column
* Filled missing values using **median rating within each product category**

---

### 🔹 Data Cleaning

* Converted column names into **snake_case** for better readability
* Renamed columns for SQL compatibility

---

### 🔹 Feature Engineering

* Created **Age Group** (Young Adult, Adult, Middle-aged, Senior)
* Converted purchase frequency into **numeric days** for analysis
* Identified and removed redundant column (**Promo Code Used**)

---

# 🗄️ Database Integration

* Connected Python to multiple databases:

  * MySQL
  * PostgreSQL
  * SQL Server
* Loaded cleaned dataset into database tables for analysis

---

# 📊 SQL Analysis

### 🔹 Revenue Analysis

* Compared total revenue generated by male vs female customers

### 🔹 Discount Behavior

* Identified customers who used discounts but spent more than average

### 🔹 Product Performance

* Top 5 products based on average review rating
* Top 3 most purchased products within each category

### 🔹 Shipping Analysis

* Compared average purchase amounts between different shipping types

### 🔹 Subscription Analysis

* Compared spending behavior between subscribed and non-subscribed customers

### 🔹 Customer Segmentation

Segmented customers into:

* New
* Returning
* Loyal

---

# 📈 Power BI Dashboard

* Connected MySQL database to Power BI using ODBC
* Built interactive dashboard including:

  * Revenue by Gender
  * Sales by Category
  * Discount vs Non-discount purchases
  * Customer segmentation
  * Product performance

---

# 🚧 Challenges & Solutions

* Faced MySQL authentication issues while connecting to Power BI
* Resolved by changing authentication method
* Used ODBC connection for stable integration

---

# 🎯 Key Insights

* Subscription customers tend to spend more
* Discounts significantly influence purchasing behavior
* Loyal customers contribute the highest revenue
* Certain product categories dominate sales
